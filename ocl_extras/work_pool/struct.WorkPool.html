pub struct WorkPool { /* fields omitted */ }

A general purpose work completion pool.

Contains elements of a single-threaded event loop and a thread pool.

Runs in and manages its own threads. Dropping the WorkPool will block the dropping thread until all submitted and spawned work is complete.
